MathML

MathML (Mathematical Markup Language = Lenguaje de Marcas Matemático) es un lenguaje de marcas dirigido a la representación de fórmulas matemáticas.

Se han publicado varias versiones de MathML: MathML 1.0 (1998), MathML 1.01 (1999), MathML 2.0 (2001), MathML 2.0 (2ª edición) (2003) y MathML 3.0 (octubre de 2010). Véase la lección sobre Historia de la Web para más detalles.

Páginas web con MathML

MathML

En HTML 5 se pueden incluir directamente elementos MathML. El problema es que actualmente (mayo de 2017) únicamente Firefox y Safari admiten MathML y ni Internet Explorer ni Chrome tienen planeado admitirlo.

Se puede comprobar fácilmente la capacidad de un navegador de mostrar elementos MathML visitando la página http://eyeasme.com/Joe/MathML/MathML_browser_test.html, que muestra varias fórmulas matemáticas tal y como se ven en un sistema de tipografía TeX, en Firefox con las fuentes STIX y en el navegador del usuario.

En los navegadores que no son capaces de mostrar elementos MathML, la solución clásica ha sido siempre mostrar las fórmulas matemáticas como imágenes, lo que impide aprovechar las posibilidades de MathML (hojas de estilos, zoom, facilidad de edición, etc.). Actualmente, la solución más habitual es utilizar alguna biblioteca JavaScript como MathJax para suplir las carencias del navegador, como por ejemplo hace la Biblioteca Digital de Fórmulas Matemáticas del NIST.

MathML en Firefox

Firefox es capaz de mostrar páginas con elementos MathML, aunque es necesario instalar ciertas fuentes para poder mostrar correctamente todos los caracteres matemáticos.

Actualmente (mayo de 2017) las fuentes recomendadas son las fuentes STIX, una fuentes de miles de caracteres creadas por el consorcio STI Pub, formado en 1995 por varias organizaciones científicas norteamericanas y la editorial Elsevier. En el año 2000 comenzó la creación de las fuentes, en mayo de 2010 se publicó la versión 1.0, en agosto de 2012 se publicó la versión 1.1 y en diciembre de 2016 se publicó la versión 2.0.

Las fuentes STIX están disponibles en formato OpenType. Actualmente (mayo de 2017), la última versión estable disponible es la versión 2.0 (de diciembre de 2016).

En cdlibre.org hay una sección dedicada a fuentes OTF, con información detallada sobre las últimas versiones publicadas.

Al abrir un archivo de extensión .xhtml como archivo local (sin utilizar un servidor de páginas web), Firefox reconoce estos documentos como application/xhtml+xml y los muestra correctamente.

MathML en Internet Explorer

Internet Explorer no es capaz de mostrar elementos MathML, pero existen plug-in gratuitos para conseguirlo, por ejemplo MathPlayer de DesignScience, cuya versión MathPlayer 4.0 se publicó en enero de 2016.

MathML en Chrome

Actualmente (mayo de 2017) Chrome no es capaz de mostrar páginas con elementos MathML y Google ha manifestado que no entra en sus planes hacerlo.

Chrome admitió MathML en la versión 24 (enero de 2013), pero debido a problemas de incompatibilidades entre el código de MathML y el de WebKit, en la versión Chrome 25 (febrero de 2013) se volvió a deshabilitar el soporte de MathML.

Edición de MathML

  • La extensión FireMath permite crear fórmulas MathML desde Firefox. Se puede instalar desde Firefox Add-ons. Actualmente se encuentra en desarrollo.
  • Web Equation: http://webdemo.myscript.com/#/demo/equation. Permite dibujar una ecuación y obtener el código MathML correspondiente.
  • Detexify: http://detexify.kirelabs.org/classify.html. Permite dibujar un símbolo matemático y obtener el código en LaTeX.
