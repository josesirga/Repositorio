Recomendaciones CSS 3

Así como CSS 2.1 es una única recomendación, para la elaboración de CSS 3 el W3C eligió un enfoque modular: CSS 3 está dividida en numerosas recomendaciones que abordan cada una un aspecto distinto de las hojas de estilo.

Cada módulo CSS 3 sigue su propio ritmo de desarrollo y los navegadores también siguen su propio ritmo de implementación de las recomendaciones o de sus borradores. En la web del W3C se pueden consultar:

  • un listado detallado de las recomendaciones CSS vigentes y en elaboración
  • listado breve de las recomendaciones CSS vigentes y en elaboración
  • listado detallado de los borradores

Muchas propiedades que aparecen en los borradores de las futuras recomendaciones de las hojas de estilo CSS 3 ya funcionan en las versiones actuales de los navegadores, pero no lo hacían en las versiones antiguas (especialmente en Internet Explorer 11, el único navegador antiguo que todavía tiene actualmente -septiembre de 2017- una cuota de mercado significativa). Por ello, al diseñar un sitio web, el diseñador debe tener en cuenta los navegadores utilizados por los usuarios de ese sitio y decidir en qué momento puede dejar de preocuparse por una determinada versión de un navegador.

Recomendaciones CSS 3 ya aprobadas

Actualmente (marzo de 2017), el W3C ha aprobado las siguientes recomendaciones CSS 3:

  • junio 2011: CSS 3: Color
  • septiembre 2011: CSS Espacios de nombres (CSS Namespaces)
  • septiembre 2011: CSS 3: Selectores
  • junio 2012: Media Queries
  • noviembre 2013: CSS 3: Atributos style (CSS Style Attributes)

Notas de grupo

Algunos documentos se han publicado como notas de grupo de trabajo (Working group notes) en vez de como recomendaciones. Eso quiere decir que el grupo de trabajo correspondiente considera que no vale la pena seguir trabajando en ese tema y publica su último borrador como nota de grupo de trabajo.

  • octubre 2014: CSS: Pefil TV 1.0 (CSS TV Profile 1.0 )
  • octubre 2014: CSS: Perfil móvil (CSS Mobile Profile 2.0)
  • octubre 2014: CSS3: Marquee (CSS Marquee Module Level 3)
  • marzo 2015: CSS Template Layout Module

Actualmente (marzo de 2017), el W3C está elaborando un gran número de recomendaciones CSS 3. Por su parte, los navegadores están implementando muchas de estas recomendaciones.
