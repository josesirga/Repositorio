CSS 2.1: Propiedades obsoletas

CSS 2 obsoletas

  • font-size-adjust
  • font-stretch
  • marker-offset
  • Colores del sistema

En esta página se comentan:

  • propiedades que estaban incluidas en la recomendación CSS 2 pero que no están incluidas en la recomendación CSS 2.1.
  • propiedades que están incluidas en la recomendación CSS 2.1, pero que se consideran obsoletas y cuyo uso no se recomienda.

Tipo de letra. Ajuste de tamaño: font-size-adjust

font-size-adjust

La propiedad font-size-adjust se introdujo en la recomendación CSS 2, pero no está incluida en la recomendación CSS 2.1, por lo que se puede considerar obsoleta en CSS 2.

Sin embargo, la propiedad font-size-adjust está incluida en la futura recomendación CSS 3: Fuentes, actualmente (mayo de 2017) en elaboración, por lo que esta propiedad se describe en la lección CSS 3: Tipo de letra (fuente).

Tipo de letra. Anchura: font-stretch

font-stretch

La propiedad font-stretch se introdujo en la recomendación CSS 2, pero no está incluida en la recomendación CSS 2.1, por lo que se puede considerar obsoleta en CSS 2.

Sin embargo, la propiedad font-stretch está incluida en la futura recomendación CSS 3: Fuentes, actualmente (mayo de 2017) en elaboración, por lo que esta propiedad se describe en la lección CSS 3: Tipo de letra (fuente).

Listas. Distancia del marcador: marker-offset y display: marker

marker-offset

La propiedad marker-offset y el valor marker de la propiedad display se introdujeron en la recomendación CSS 2 (marker-offset) y (marker), pero no están incluidos en la recomendación CSS 2.1, por lo que se pueden considerar obsoletos en CSS 2.

La propiedad marker-offset tampoco está incluida en la futura recomendación CSS 3: Listas, actualmente (mayo de 2017) en elaboración, aunque esta recomendación introduce el pseudo-elemento ::marker para establecer el estilo del marcador.

La propiedad marker-offset debería establecer la distancia del marcador al elemento de la lista. Esta propiedad afectaría a los marcadores creados con los pseudo-elementos :before y :after.

Los navegadores no son capaces de interpretar esta propiedad, ni probablemente lo hagan nunca.

li:before {
  content: "***";
  display: marker;
  width: 6em;
  text-align: center;
}

ul {
  list-style-type: none;
}

li:before {
  content: "***";
  display: marker;
  marker-offset: 40px;
}

Colores del sistema

Los colores del sistema hacen referencia a los colores utilizados por el interfaz del sistema operativo. Formaban parte de la recomendación CSS 2, pero en la recomendación CSS 3:Color se consideran obsoletos, en favor de la propiedad appearance, definida únicamente para elementos de formularios en la futura recomendación CSS 4: Interface de usuario básico, actualmente (octubre de 2017) en elaboración.
