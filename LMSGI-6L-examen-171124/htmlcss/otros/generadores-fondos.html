En esta lección se comentan páginas web y aplicaciones que facilitan la creación de imágenes de fondo.

Repostorios de imágenes

En la web se pueden encontrar muchos repositorios de imágenes para fondos ya diseñadas y listas para utilizar. Normalmente, las imágenes se encuentran en formato JPG o PNG, pero en algunos casos las imágenes se pueden descargar en formato SVG (u otros formatos convertibles a SVG), lo que permite su modificación y reutilización posterior.

  • Open Clip Art Library: https://openclipart.org/

    Dibujos vectoriales en formato SVG y PNG, con licencia dominio público.

  • Vecteezy: https://www.vecteezy.com/

    Dibujos vectoriales en diferentes formatos vectoriales. hay tanto imágenes libres como de pago.

    El formato de las imágenes no suele ser SVG, sino formatos propios de aplicaciones propietarias como Adobe Illustrator o más o menos abiertos (eps, etc.).

  • Colrd

    Repositorio de colores, paletas de colores, gradientes e imágenes de fondo creadas y compartidas por usuarios.

Aplicaciones de escritorio

El editor gráfico Inkscape permite crear y modificar imágenes vectoriales que se pueden utilizar como fondos. En la lección Inkscape se comenta cómo conseguir y utilizar este programa.

Aplicaciones web

Editores gráficos

  • SVG-edit, una aplicación web que se puede probar en la web.
  • Vectr, otra aplicación web para editar dibujos SVG.

Convertidores

  • cloudconvert

    Permite convertir entre numerosos formatos de archivos, incluidos formatos gráficos vectoriales (ai, cdr, eps, svg, vsd, etc.). Tiene un límite diario de conversiones gratuitas.

  • SVGOMG! (SVGO Missing GUI)

    Optimizador de imágenes SVG. Reduce el tamaño de las imágenes SVG. Es una interfaz gráfica del proyecto SVGO (SVG Optimizer).

Generadores

  • Para crear un patrón de fondo píxel a píxel:
    • BGMaker
    • patternify
  • Para generar imágenes de franjas:
    • Stripe Generator
    • stripemania
    • Striped Backgrounds - franjas verticales de cinco colores
  • Para generar imágenes sencillas:
    • Tartan Designer - imágenes similares a tejidos
    • Pixelknetes - imágenes de puntos
    • Noise Texture Generator - imágenes de color plano con ruido.
  • Para generar imágenes con texturas:
    • ZenBG - texturas y gradientes de colores
    • PatternCooler - cientos de patrones ya diseñados y personalizables
  • Editores que permiten combinar imágenes:
    • Pattern Ninja
    • BGPatterns - Versión antigua de Pattern Ninja.
    • COLOURlovers
