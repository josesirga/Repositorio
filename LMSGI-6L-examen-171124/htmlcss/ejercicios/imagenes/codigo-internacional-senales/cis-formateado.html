El código internacional de señales

El código internacional de señales es un sistema de señales y códigos diseñado para ser utilizado por barcos para comunicar mensajes importantes relacionados con la seguridad en la navegación. Las señales se pueden transmitir utilizando banderas, señales luminosas, radiotelegrafía o radiotelefonía.
